This is the Sauerbraten 3D FPS compiled from C++ and OpenGL to JavaScript and WebGL using Emscripten.

The demo might take time to download. Afterwards, press the "fullscreen" button and then "GO!". Move with WASD, jump with space, look around with the mouse. You can shoot a little by clicking the mouse. Some editing stuff works (press 'e' and fly around).

Please note that

  • The C++ game code has not been optimized at all in any way yet
  • The generated JavaScript is itself not fully optimized yet, nor even minified
  • The level you see when you press "GO!" was made by azakai, a person with 0 artistic talent
  • The game assets (textures) have not been optimized for faster downloads at all
  • Very recent dev versions of browsers are needed to get all functionality to work, like fullscreen and pointer lock. Tested as working in Firefox 15.
So this is a very very early demo - ignore performance and content quality!
